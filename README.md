# 暗記カード PWA

スマートフォン・タブレット・PCで使えるスペースドリピティション暗記カードアプリです。

## フォルダ構成

```
暗記カード/
├── index.html          ← アプリ本体
├── manifest.json       ← PWA 設定
├── sw.js               ← Service Worker（オフライン対応）
├── css/
│   └── style.css
├── js/
│   ├── db.js           ← IndexedDB 操作
│   ├── loader.js       ← Excel/CSV 読み込み
│   ├── sm2.js          ← SM-2 アルゴリズム
│   ├── session.js      ← 学習セッション管理
│   ├── ui.js           ← UI ヘルパー
│   └── app.js          ← アプリ全体制御
├── data/
│   └── 暗記カード.xlsx ← 問題データ（参考用）
└── images/
    ├── icon-192.png
    └── icon-512.png
```

---

## GitHub Pages へのアップロード方法

1. GitHub でリポジトリを作成
2. このフォルダの中身をすべてアップロード（`git push` または Web UI）
3. リポジトリの Settings → Pages → Source を `main` ブランチ / `/ (root)` に設定
4. `https://ユーザー名.github.io/リポジトリ名/` でアクセス

**ローカルサーバー不要・Python 不要です。**

## 使い方

### 初回
1. アプリを開く
2. 「ファイルを選ぶ」ボタンで `暗記カード.xlsx` を選択
3. データが読み込まれ、以降は自動でロードされる

### 2回目以降
アプリを開くだけで自動的に前回のデータが読み込まれます。

### 問題データを更新したいとき
1. `暗記カード.xlsx` を編集・保存
2. アプリのホーム画面で「🔄 問題データを更新する」をタップ
3. 新しい xlsx を選択 → **学習履歴は保持されます**

## スマートフォンへのインストール（ホーム画面追加）

### iPhone (Safari)
1. Safari でアプリを開く
2. 共有ボタン（□↑）→「ホーム画面に追加」

### Android (Chrome)
1. Chrome でアプリを開く
2. メニュー（⋮）→「アプリをインストール」または「ホーム画面に追加」

ホーム画面から起動するとアプリのように全画面で表示されます。

## 問題データの編集

### 列構成（1行目はヘッダー）
| 列 | 内容 | 例 |
|----|------|----|
| A  | ID   | `s1`（科目内で一意） |
| B  | 単元 | `生命：植物のつくり` |
| C  | 問題 | `おしべの先にある袋を何というか。` |
| D  | 答え | `やく（葯）` |

### 科目を追加する
**シートを追加するだけ**でOKです。HTML/JS の編集は不要。

### 単元の表示名
`大分類：小分類` 形式にすると、単元フィルターで小分類名が表示されます。

## 学習モード

| モード | 内容 |
|--------|------|
| 📅 今日の復習 | SM-2 で期限が来たカード |
| 📋 全問題 | 選択中の単元をランダム出題 |
| 💪 苦手問題 | 定着度（EF値）が低い順に出題 |
| 🎲 ランダム | 単元フィルタ対象をシャッフル |

```
↓

↓
GitHub Pages へ push
↓
PC・スマホ・タブレットすべてで同じカードを利用
```
